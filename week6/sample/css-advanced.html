CSS class 를 적용하는 여러가지 방법

1. 하나의 요소에 여러개의 class 적용하기

box1,2,3 여러개의 클래스에 동일한 스타일을 적용할 수 있다.
.box1, .box2, .box3 { width:100px;~ } 과 같은 방식으로 코드를 작성할 수 있다.
또한, class="class1 class2" 와 같이 여러개의 클래스를 중복 적용할 수 있다.
*주의할 점: 클래스명 앞에 반드시 "." 클래스 선택자에 대한 표시를 빼먹지 말 것.

2. 클래스 2개가 모두 적용된 요소를 선택하여 스타일 지정하기

.class1.class2 (중간 공백 없음)을 사용해서 class1과 class2가 "둘다" 적용된 요소를 선택하여 적용할 수 있다.
auqa-box와 box-border 두 클래스가 모두 적용된 박스.
css 코드를 확인해주세요
(검색: .aqua-box.box-border)

3. 어떤 클래스에 속한 자식 요소의 클래스를 선택하여 스타일 지정하기

.class1 (공백).class2 로 class1이 적용된 태그에 포함된 class2 요소를 선택하여 적용할 수 있다.
여기서 parent-box는 class1(부모요소), child-circle는 class2(자식요소)로 볼 수 있다.
parent-box
child-circle

*div class="child-circle"만 적용된 경우

child-circle 단독으로는 스타일이 적용되지 않음(해당부분 코드 확인 필수)
